Why Exastro ?

Exastro is an open source software suite
for digitizing, automating and labor saving
the system life cycle
(design, development, setting and operation).

Exastro Suite

  • Exastro IT Automation Setting
    Exastro IT Automation offers solutions to the following problems in the system life cycle:
    ・Data management errors
    ・Manual processing mistakes
    ・Delays in responding to problems
    Click here for details
  • Exastro Playbook Collection Setting
    Exastro Playbook Collection is a collection of Ansible Playbooks for OS and Middleware used in System Integration.
    It include playbooks that collects parameters from the actual server, and can be used to check settings after construction.
    Click here for details
  • Exastro Operation Autonomy Support Engine Operation
    Exastro Operation Autonomy Support Engine is software for realizing automation, efficiency and labor saving of system operation tasks.
    Specifically, it centrally manages know-how and documents in system operation as rules, and look for workarounds from operation information.
    And if it finds a workaround, it controls the automation softwares such as Exastro IT Automation and execute the workaround automatically.
    Click here for details
